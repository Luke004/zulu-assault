Create your custom map using the folloing properties in a map editor program called Tiled (Official Download Link). When you are finished with the map, put the created *.tmx-file in this folder ('/assets/maps_custom').

The tileset you will need to create a map in Tiled is called 'landscape_tiles.png' and can be found in this project under 'assets/tilesets'.

After map creation, you can then open your created *.tmx-file in the Level Editor in game and place all your game objects (entities, items, circles).

Thanks for playing and enjoy your custom maps :)
